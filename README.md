# 💖 เว็บไซต์เซอร์ไพรส์วันครบรอบ / วันเกิด (Romance Surprise Gift Box)

เว็บไซต์ของขวัญออนไลน์ที่ออกแบบมาอย่างประณีต โรแมนติก และอบอุ่น เพื่อเซอร์ไพรส์คนพิเศษในวันสำคัญ (เช่น วันครบรอบ, วันเกิด, หรือวันวาเลนไทน์) แยกโครงสร้างโค้ดออกเป็น `index.html`, `style.css` และ `script.js` พร้อมระบบสังเคราะห์เสียงเปียโนสุดพิเศษที่ไม่ต้องการเซิร์ฟเวอร์หรือไฟล์เพลงภายนอกในการทำงานเริ่มต้น!

---

## ✨ ฟีเจอร์หลัก (Key Features)
1. **หน้าแรก (Lock Screen):** จำลองหน้าจอปลดล็อกโทรศัพท์สวยงาม มีช่อง PIN 6 หลักและปุ่มกดตัวเลข เมื่อกดถูกจะเล่นเสียง Chime และแฟลชเอฟเฟกต์ปลดล็อก ถ้ารหัสผิดจะสั่นเบาๆ พร้อมเอฟเฟกต์การ์ดสั่นแจ้งเตือน ❤️
2. **หน้าที่สอง (Welcome):** แสดงรูปถ่ายคู่สไตล์โพลารอยด์ลอยได้เบาๆ ชื่อแฟน และข้อความซึ้งๆ พร้อมหัวใจลอยขึ้นมาจากด้านล่างเรื่อยๆ
3. **หน้าที่สาม (Memory Timeline):** เส้นสายความทรงจำแสดงเรื่องราวเรียงตามเวลาแบบ Responsive (สองคอลัมน์บน PC, หนึ่งคอลัมน์บนมือถือ) สามารถกดคลิกที่รูปภาพเพื่อขยายใหญ่ (Lightbox) พร้อมคำบรรยายสุดโรแมนติก
4. **หน้าที่สี่ (Typing Letter):** การ์ดซองจดหมาย 3D เมื่อกดเปิด ซองจดหมายจะเปิดออกและจดหมายเลื่อนขึ้นมาแสดงข้อความแบบพิมพ์ทีละตัวอักษร (Typing Effect) พร้อมเสียงแป้นพิมพ์เบาๆ และหัวใจกระจายรอบจอเมื่อพิมพ์จบ
5. **หน้าสุดท้าย (Final Surprise):** ข้อความใหญ่คำว่า "I Love You" และปุ่มรับเซอร์ไพรส์ เมื่อกดปุ่มจะแสดงการ์ดใบสุดท้ายพร้อมเอฟเฟกต์พลุไฟหลากสี กระดาษ Confetti ร่วงหล่น และหัวใจลอยฟุ้งเต็มหน้าจอแบบจุใจ!
6. **ระบบเพลง (Romantic Audio Synthesizer):** หากไม่ได้ใส่ลิงก์เพลงภายนอก ระบบจะทำการสังเคราะห์เสียงเปียโนและเมโลดี้สดๆ (Procedural Audio) ขึ้นมาเล่นเบาๆ อย่างอบอุ่นทันทีหลังปลดล็อก เพื่อความสุนทรีโดยไม่ต้องโหลดไฟล์หนัก

## 🛠️ วิธีการเปิดเล่นและทดสอบเว็บไซต์ (How to Run)
คุณสามารถเปิดเล่นเว็บไซต์นี้ผ่านคอมพิวเตอร์และมือถือได้ง่ายๆ:
1. **ใช้ VS Code (แนะนำ):**
   - เปิดโฟลเดอร์ `surprise-website` ใน VS Code
   - ติดตั้ง Extension ชื่อ **Live Server**
   - คลิกขวาที่ไฟล์ `index.html` แล้วเลือก **Open with Live Server**
   - เว็บไซต์จะเปิดขึ้นบนเบราว์เซอร์อัตโนมัติ และรองรับการพรีวิวแบบเรียลไทม์เมื่อมีการปรับปรุงโค้ด
2. **เปิดไฟล์ตรงๆ:**
   - ดับเบิ้ลคลิกไฟล์ [index.html](file:///C:/Users/Nitiwat/surprise-website/index.html) ในเครื่องของคุณได้เลย!

## 🎨 วิธีปรับแต่งเนื้อหา (How to Customize)
คุณสามารถปรับแต่งรหัสผ่าน รูปภาพ ข้อความ และเพลงได้ง่ายๆ โดยแก้ไขส่วนหัวของไฟล์ [script.js](file:///C:/Users/Nitiwat/surprise-website/script.js) ในจุดที่เขียนว่า `CONFIG` ดังนี้:

### 1. เปลี่ยนรหัสผ่านปลดล็อก (PIN Code)
แก้ไขบรรทัด:
```javascript
PIN_CODE: "210625", // เปลี่ยนเป็นวันเกิด, วันครบรอบ (ววดดปป หรือตามใจชอบ) เช่น "250668"
```

### 2. เปลี่ยนชื่อแฟนและข้อความ
แก้ไขบรรทัด:
```javascript
PARTNER_NAME: "น้องน้ำหวาน ❤️",
WELCOME_TITLE: "สุขสันต์วันครบรอบนะคนเก่ง ❤️",
WELCOME_MESSAGE: `ขอบคุณที่เข้ามาเป็นส่วนหนึ่งของชีวิต...`,
LOVE_LETTER: `ถึงคนที่รักที่สุด...`,
```

### 3. เปลี่ยนรูปภาพ (Images)
คุณสามารถใช้ลิงก์รูปภาพออนไลน์ หรือนำรูปภาพของคุณมาเซฟลงในโฟลเดอร์โปรเจกต์ เช่น สร้างโฟลเดอร์ย่อย `images/` แล้วเปลี่ยนจุดเชื่อมโยงดังนี้:
```javascript
AVATAR_IMAGE: "images/avatar.jpg",   // รูปหน้าจอ Lock Screen
WELCOME_IMAGE: "images/welcome.jpg", // รูปหน้าพรีวิวแรก
FINAL_IMAGE: "images/together.jpg"   // รูปหน้าสุดท้าย
```

### 4. เพิ่มรูปภาพความทรงจำใน Timeline (Memories Gallery)
คุณสามารถเพิ่มหรือลดรูปภาพใน Timeline ได้ไม่จำกัด โดยการเพิ่ม/แก้ไขสมาชิกในอาร์เรย์ `MEMORIES` ตามรูปแบบนี้:
```javascript
MEMORIES: [
    {
        date: "14 กุมภาพันธ์ 2024",
        image: "images/memory1.jpg",
        caption: "เดทแรกของเราที่ร้านอาหารริมน้ำ ❤️"
    },
    // สามารถเพิ่มรูปภาพถัดๆ ไปได้ตามรูปแบบด้านบน
]
```

### 5. ใส่เพลงรักที่คุณชอบ (Background Music)
หากต้องการใส่เพลงส่วนตัว ให้นำไฟล์เพลงประเภท `.mp3` ไปวางไว้ในโปรเจกต์ หรือนำ URL เพลงออนไลน์มาเติมลงในช่องนี้:
```javascript
AUDIO_URL: "music/our-song.mp3" // หากปล่อยว่างไว้ ระบบจะเล่นเพลงเปียโน Ambient อัตโนมัติ
```

## 🔒 ฟังก์ชันรีเซ็ตและตัวช่วยจำปลดล็อก (LocalStorage)
- เว็บไซต์ใช้ **LocalStorage** เพื่อช่วยจำสถานะการปลดล็อก หากคนพิเศษของคุณเปิดมาครั้งแรกแล้วผ่านรหัสผ่านไปได้แล้ว เมื่อปิดเบราว์เซอร์ไปแล้วเปิดขึ้นมาใหม่ในครั้งถัดไป ระบบจะจดจำและผ่านหน้า Lock Screen เข้าไปพรีวิวความรักได้ทันที
- หากคุณต้องการทดลองความรู้สึกเหมือนล็อกหน้าจอใหม่อีกรอบ สามารถเลื่อนลงไปหน้าสุดท้ายแล้วกดปุ่ม **"ล็อคหน้าจอ 🔒"** เพื่อเคลียร์ความจำและกลับไปหน้าใส่รหัสผ่านได้ทุกเมื่อ
